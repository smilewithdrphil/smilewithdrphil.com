WarpSpeed by SWDP

WarpSpeed by SWDP is a Chrome extension built for ClinCheck that transforms how you write treatment instructions. Instead of typing full clinical phrases manually, WarpSpeed lets you enter short commands directly in the comment box. Type a command like TI8M10, press Enter, and it instantly expands to a properly formatted instruction ready for your technician. Built because my own ClinCheck workflow demanded it.

Try WarpSpeed →

SWDP Toolkit

SWDP Toolkit is a Chrome extension that adds the clinical workflow tools I actually use during every ClinCheck review — patient date of birth and age displayed automatically, a case timer that copies your comment and case link on stop, and a fully digital version of my ClinCheck review checklist. Built because I kept a physical checklist and finally decided to build a better one.
